Module builtin

The builtin module contains functions / constants that are available by default in every Erg program. These functions provide basic functionality for input/output, error handling, and other common tasks.

Subroutines

print!

The print function is used to output text or values to the console. It takes one or more objects as arguments and displays their string representation. The objects are separated by spaces and a newline character is added at the end.

Type

(*objects: Ref(Obj), sep := Str, end := Str, file := Writable!, flush := Bool) => NoneType

Example

                        print! "Hello, world!"
print! "Hello", "world!", sep := ", "

Output

Hello, world!
Hello, world!
