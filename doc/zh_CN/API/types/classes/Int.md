表示整数的类。实例为 0、1、-1、300000 等
很多语言即使在表示自然数的时候也使用Int(类型等价于)，但是Erg使用了使用更小的类型的原则，
建议使用 Nat、NZInt、Interval 类型等
Int是一种来自 Python 的类型，只有`Object`作为其超类型
